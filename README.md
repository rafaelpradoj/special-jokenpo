# Rock Paper Scissors Lizard Spock (Python)

Um jogo clássico de Jokenpô expandido, desenvolvido em Python para praticar lógica de programação, estruturas de dados e controle de fluxo.

## 🚀 Como rodar
1. Certifique-se de ter o [Python](https://www.python.org/) instalado no seu sistema.
2. Clone este repositório:
   `git clone https://github.com/rafaelpradoj/special-jokenpo`
3. Navegue até a pasta:
   `cd jokenpo`
4. Execute o jogo:
   `python3 app.py`

## 🛠️ Tecnologias e Conceitos aplicados
- Linguagem: Python
- Estruturas: `while`, `try/except`, dicionários, `random`, `time`.
- Validação de entrada de dados e tratamento de erros.

## 💡 O que aprendi
Este projeto foi fundamental para entender a importância de:
- Manter o código limpo (DRY - Don't Repeat Yourself).
- Validar entradas do usuário para evitar crashes.
- Utilizar dicionários para mapeamento de dados.

## 📝 Diário de Aprendizado (Roadmap)
